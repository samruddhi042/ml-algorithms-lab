# ML Algorithms Lab 🤖

A collection of Machine Learning algorithm implementations and experiments using Python and Google Colab.

## 📁 Contents
- Jupyter/Colab notebooks for various ML algorithms
- Datasets used for experimentation
- Model training and evaluation scripts



## 🚀 How to Run
1. Clone the repo
```bash
   git clone https://github.com/samruddhi042/ml-algorithms-lab.git
```
2. Open any `.ipynb` file in Google Colab or Jupyter Notebook
3. Run cells sequentially
